Membuat Model

Membuat Model

Ikuti langkah-langkah berikut untuk membuat model dari response REST API:

  • Bukalah url berikut https://api.chucknorris.io/jokes/random dengan web browser anda.
  • Salinlah response tersebut yang berupa format JSON.
  • Bukalah web http://www.jsonschema2pojo.org/ yang digunakan untuk membangkitkan POJO (Plain Old Java Object) dari format JSON.
  • Tempel salinan response API ke dalam inputan.
  • Sesuaikan package dengan package project android anda, dan tambahkan model.
  • Kemudian beri nama Class name dengan Joke
  • Pilih Source type sebagai JSON
  • Pada project ini digunakan library Gson, sehingga sesuaikan Annotation style menjadi Gson
  • Anda dapat menggunakan tombol Preview kemudian menyalin isi kode Java atau bisa mengunduh dengan format zip, dengan menekan tombol Zip
  • Dengan langkah ini anda telah mendapatkan model yang sesuai dengan response API yang dibutuhkan secara otomatis.
